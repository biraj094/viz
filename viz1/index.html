How to Use NepalScope

Choose a Layer

Use the header buttons to set the granularity — from 5 Development Regions down to 36,000+ Wards. The map reloads with the selected administrative level.

Select & Compare

  • Click a region to set it as your reference
  • Hover over any other region to compare areas in real time
  • Click while hovering to freeze the comparison

Insights Panel

The right panel shows each region's area, its rank among all regions, its parent province or district, and a size ratio when two regions are compared.

Reset

Click Reset to clear the current selection and start a fresh comparison.
