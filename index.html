Bigha Busters

Nepal Streamlit Python Land Conversion rupantaran

Ever wondered how many basketball courts fit in your plot, or whether your backyard is bigger than the Great Pyramid's base? Built using rupantaran, an open-source Python package for Nepali measurement conversions (docs).
